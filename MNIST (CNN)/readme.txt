This is a Convolutional Neural Network model which is used for Hand-written Digits Classifier.
The program is written with Python, using Pytorch framework

Datasets: MNIST - 42000 picture samples - dimensions 28x28pixels
Framework: Pytorch
ML Algorithm applied:
	Model: CNN (max pooling, ReLU)
	Optimizer: Adam
	Loss: Cross-Entropy
	Other: K-Fold Cross-Validation, Bounding Box, L2 Regularization, GPU
