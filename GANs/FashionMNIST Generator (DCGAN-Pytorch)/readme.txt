This is a Generative Adversarial Networks model which is used for generate samples from Fashion MNIST.
The program was written with Python, used Pytorch framework

Datasets: FashionMNIST - 60000 picture samples - dimensions 28x28pixels
Framework: Pytorch
ML Algorithm applied:
	Model: DCGANs (added one-sided label smoothing, weight initiation)
	Optimizer: Adam
	Loss: Cross-Entropy
	Other: GPU
